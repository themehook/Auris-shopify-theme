# Blueprint — Luno Theme Reference

This folder stores skill files for every custom section and feature built or modified in this theme. Each file captures the full requirements, design decisions, schema structure, and technical context needed to make future updates without re-researching from scratch.

## Start here every session

**Read [`conversation-log.md`](conversation-log.md) first.** It contains the full record of all sessions, requests, decisions, and outcomes. After completing work, append a new session entry to that file.

## Files

| File | Purpose |
|------|---------|
| [conversation-log.md](conversation-log.md) | **Running session log** — read first, update after every session |
| [theme-conventions.md](theme-conventions.md) | Shared patterns used across all sections |
| [interactive-feature-showcase.md](interactive-feature-showcase.md) | Interactive Feature Showcase section — full spec |
| [scroll-reveal-media.md](scroll-reveal-media.md) | Scroll Reveal Media section — updates made |
| [tab-collage.md](tab-collage.md) | Tab Collage — original design + abandoned Style 2 history |
| [slideshow.md](slideshow.md) | Slideshow section — Feature List spec (title, font size, icon toggle, schema, rendering) |

## Workflow

1. **Read** `conversation-log.md` — get context from the last session.
2. **Read** the matching section skill file before editing that section.
3. **Check** `theme-conventions.md` for shared patterns (color schemes, heading technique, etc.).
4. **After work:** Append a new entry to `conversation-log.md` and update the relevant skill file if anything changed.
